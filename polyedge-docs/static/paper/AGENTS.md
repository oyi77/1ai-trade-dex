# polyedge-docs/static/paper

## Purpose
Research paper assets served via the Docusaurus site. Contains the compiled paper PDF, supplementary PDF, and abstract video.

## Key Files

| File | Description |
|------|-------------|
| `paper.pdf` | Compiled research paper PDF |
| `supplementary.pdf` | Supplementary materials PDF |
| `abstract_video.mp4` | 50-second abstract overview video |

## For AI Agents

### Working In This Directory
- Source LaTeX is in `docs/paper/` -- rebuild PDFs there, then copy here
- Video generated by `docs/paper/scripts/generate_abstract_video.py`
- Keep file sizes in mind -- these are served to end users

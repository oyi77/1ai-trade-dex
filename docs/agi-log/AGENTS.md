# agi-log

## Purpose
Daily AGI log outputs. The `nightly_review` job generates a markdown log here each day with strategy performance, calibration analysis, and improvement plans.

## Key Files

| File | Description |
|------|-------------|
| `2026-05-03-diagnostic.md` | Diagnostic log |
| `2026-05-04.md` | Daily AGI review log |
| `2026-05-06.md` | Daily AGI review log |
| `2026-05-10.md` | Daily AGI review log |

## For AI Agents

### Working In This Directory
- Generated automatically by `backend.core.nightly_review` (controlled by `AGI_NIGHTLY_REVIEW_ENABLED`)
- Do NOT manually edit — these are audit logs
- Format: date-based markdown files (e.g., 2026-05-10.md)

### Common Patterns
- Read by dashboard and monitoring tools
- Contains daily strategy health, calibration, and improvement recommendations

## Dependencies

### Internal
- Generated by `backend.core.nightly_review`
